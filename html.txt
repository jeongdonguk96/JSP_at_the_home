GET방식
  정보를 숨기지 않고 넘겨줌
POST방식
  정보를 숨긴채 넘겨줌

form
  method get방식과 post방식 둘 중 하나 선택
  
  fieldset 칸을 할당
  legend 칸의 이름
  lable input에 이름지어주는 것 + for을 사용해 input과 연결시켜 줘야함
        혹은 input을 lable안에 넣으면 for없이도 연결로 인식
  type input의 타입을 지정 - text, radio, checkbox, submit 등
        비슷하게 보이지만 타입을 강제해줘서 형식에 맞지 않으면 걸러내줌
  name 변수명으로 사용, 중복가능 (내용을 설명하는 역할로)
  id 고유한 명칭 (구분자로써 역할)
  size 크기 조정
  autofocus 자동으로 입력포커스가 할당 (아이디에 커서찍는 기능)
  autocomplete 커서를 누르거나 아래키 화살표를 누르면 지난 검색기록 보여줌
  placeholder 입력창에 희미한 글자로 설명
  required 필수 입력 (*의 역할)
